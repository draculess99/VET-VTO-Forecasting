AI-Powered Warehouse Workforce Forecasting and VET/VTO Optimization

This project demonstrates an end-to-end warehouse workforce forecasting and VET/VTO decision support system. It uses historical demand data and machine learning to forecast weekly workload, recommend VET, VTO, or Normal staffing actions, compare model performance against baseline forecasting methods, and estimate potential staffing cost impact.

Project Highlights

  • XGBoost-based weekly workload forecasting
  • VET/VTO/Normal staffing recommendation logic
  • Streamlit dashboard and Flask API architecture
  • Docker containerization
  • AWS ECS Fargate deployment evidence
  • Clean portfolio-ready GitHub structure
