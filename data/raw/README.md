# Data Directory

This folder contains the input data used for the warehouse workforce forecasting project.

The project uses Walmart weekly sales data as a proxy for warehouse workload demand. The raw dataset should be placed in the `data/raw/` folder, while cleaned or processed files can be stored in `data/processed/`.

Large data files may be excluded from GitHub using `.gitignore`. If the raw data is not included, download the Walmart dataset from Kaggle and place the files in the appropriate folder before running the notebook.
